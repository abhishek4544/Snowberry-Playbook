# Component Rules

Before creating any UI:

1. Search `COMPONENT_LIBRARY.md`
2. Reuse an existing component
3. Never create a new component if an existing one solves 80% of the requirement
4. If a new component is needed, provide the following before creating it:
   - **Reason** — why no existing component works
   - **Usage** — where and how it will be used
   - **Variants** — what states/sizes/colors it needs
   - **Figma location** — the exact Figma node before writing any code
