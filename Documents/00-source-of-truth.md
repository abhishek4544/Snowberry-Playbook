# Source of Truth

## Rule

The Figma file is the primary design source of truth for all Snowberry work.

**Figma file:** https://www.figma.com/design/65cijwcIaXfRuKxnUKwJlE/Snowberry-Web-app
**Components page:** https://www.figma.com/design/65cijwcIaXfRuKxnUKwJlE/Snowberry-Web-app?node-id=2556-6455

## Priority Order

```
Figma > Documentation > Any other source
```

## Enforcement Rules

1. **Conflict resolution** — If Figma and any document disagree, Figma wins. Update the doc, never the design.
2. **No new components** — Never create or propose a new component if a similar one already exists in Figma. Inspect first.
3. **Always inspect Figma first** — Before any component work, implementation, or documentation, pull the current state from the Figma file.
4. **DESIGN_SYSTEM.md is derived** — It reflects Figma, it does not define it. Keep it in sync, never treat it as authoritative.
